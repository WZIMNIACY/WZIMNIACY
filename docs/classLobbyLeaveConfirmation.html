Detailed Description

Obsługuje wyświetlanie okna potwierdzenia przy opuszczaniu lobby i wywołuje EOSManager.LeaveLobby przed zmianą sceny.

Zakłada obecność autoloadu EOSManager. Dialog tworzony dynamicznie w wątku głównym; klasa nie jest thread-safe.
