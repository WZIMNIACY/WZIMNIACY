Detailed Description

Obsługuje cofanie się do poprzedniej sceny za pomocą klawisza ESC. Nie działa gdy jakiś input jest aktywny (LineEdit, TextEdit, itp.). Może korzystać z LobbyLeaveConfirmation oraz opuszczać lobby przez EOSManager.LeaveLobby.

Wymaga autoloadu EOSManager i (opcjonalnie) instancji LobbyLeaveConfirmation w scenie; logika powinna działać w wątku głównym Godota i nie jest thread-safe.
